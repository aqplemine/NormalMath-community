# Contributing to NormalMath

If you want to make NormalMath better in any way, follow the
instructions below to make your ideas come to life faster!

## Pre-requisites

- You need to have a [GitHub account][createaccount] to make contributions.
- Your GitHub account must be elegible for the following:
  - Must not be a bot (unless manually used to make 
  - Not be banned/marked as spam by GitHub
  - Not be recognized by any NormalMath staff as breaking GitHub or HLIWTPH or Idiot Studios or NormalMath's TOS/Rules/Privacy Policy.
- You accept these rules here, accept GitHub, HLIWTPH, Idiot Studios and NormalMath's TOS/Rules/Privacy Policy.
  - > Idiot Studios' TOS/Rules/Privacy Policy is unavaliable as of now.
    
    > HLIWTPH's Tos/Rules/Privacy Policy us unavaliable as of now.
    
    > NormalMath's Rules (partially)/Privacy Policy (a very slight amount avaliable,) are unavaliable right now.
    
    NormalMath's TOS: [NormalMath TOS][normalmathtos]
    You may visit any GitHub resource to see their TOS/Rules/Privacy Policy.
